# Netwerk in Kaart
Een lichte, webgebaseerde tool om netwerkpartners in zorg- en welzijnsregio’s inzichtelijk te maken.

- Ontwikkelaar: Sander Knoester  
- Technologie: HTML, CSS, JavaScript, PNG-assets  
- Huidige status: Prototype  
- Domein: [https://netwerkinkaart.nl](https://netwerkinkaart.nl)  
- Doel: Transparantie, samenwerking en gebruiksgemak binnen publieke ketennetwerken.
